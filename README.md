# 이커머스 고객 세분화 및 마케팅 전략 제안

레드오션 이커머스 시장에서 핵심·잠재 고객을 식별하고, 고객 행동 데이터 기반의
세분화·맞춤형 마케팅 전략을 도출하는 데이터 분석 프로젝트.

거래·고객·쿠폰·마케팅·세금 데이터를 통합해 RFM 세그먼테이션, 코호트·리텐션
분석, 쿠폰 효과의 인과추론, A/B 테스트 설계, Tableau 대시보드까지 수행.

## 진행 기간
2025.03 ~ 진행 중

## 역할
- 2025.03 — 4인 팀, 기여도 30% (EDA·데이터 전처리·결과 해석·시각화)
- 2026.05 ~ 진행 중 — 단독 (SQL 재구축, 인과추론, A/B 설계, BI 대시보드)

## 데이터 출처
데이콘 이커머스 고객 세분화 분석 아이디어 경진대회 (2024)
- 거래 52,924건, 고객 1,468명, 2019년 1년치
- 5개 테이블: Onlinesales, Customer, Discount, Marketing, Tax

## 분석 흐름

### 데이터 처리
- 5개 데이터셋 병합 및 결측치 처리
- 분석용 파생 변수 생성 (고객 총 구매금액·수량·방문횟수)
- BigQuery SQL로 재구축 (CTE, NTILE, 윈도우 함수)

### 탐색적 분석 (EDA)
- 성별·지역·제품 카테고리별 구매 분포
- 재구매 여부에 따른 구매 패턴·쿠폰 사용 비교
- 마케팅 비용 추이와 구매 행동 간 관계

### RFM 세그먼테이션
- Recency·Frequency·Monetary 지표 산출 및 분위 점수화
- 11개 초기 세그먼트를 5개 핵심 세그먼트로 재구성:
  핵심 파트너 / 성장형 / 유망 / 이탈 위험 / 장기 비활성

### 고객 행동 분석
- 거래 기반 5단계 Funnel 전환율
- 코호트·리텐션 매트릭스 (가입월 × 월차)
- 쿠폰 사용과 리텐션 간 관계 (카이제곱·피어슨)
- PSM 기반 인과 효과 측정 (예정)

### 전략 제시
- 핵심 파트너: VIP 집중 관리, 연말·주중 타겟 캠페인
- 성장형·유망: 개인화 쿠폰, 전환 유도 마케팅
- 이탈 위험·장기 비활성: 긴급 리텐션·재활성화 전략

## 주요 결과

![Phase 2 핵심 발견 8개](visualizations/phase2_summary.png)

**Phase 1 — 세그먼테이션 · Funnel**
- 5개 세그먼트 중 핵심 파트너 + 성장형이 전체 매출의 약 68~70% 차지
- Funnel 가장 큰 비즈니스 이탈 지점: 첫 구매 → 재구매 단계 8.51%
- 쿠폰 사용과 리텐션 간 양의 상관관계 확인 (피어슨 r = 0.521)

**Phase 2 — 코호트 · LTV · 재구매 간격**
- 1개월차 평균 리텐션 **9.68%** — 가입 직후 90% 이탈, Funnel 결과를 코호트 관점에서 재확인
- 1월 코호트 m5~m7 **20%대 중기 반등** — 코호트 곡선의 단조 감소 가정이 깨지는 사례
- 핵심 파트너 LTV가 **나머지 4개 세그먼트 합산의 3배** — RFM 분류의 실제 가치 집중도
- 장기 비활성 세그먼트는 **거액 단발 이탈자** (짧은 유지기간 + 높은 ARPU의 ARPU 역설)
- 세그먼트별 마케팅 발송 주기 baseline (IPT p50): 핵심 파트너 **27일** · 성장형 **36일** · 유망 **54일**
- 8개 발견 정리: [reports/cohort_ltv_findings.md](./reports/cohort_ltv_findings.md)
